# us-stock-app-public
Public repository to fetch the us-stock-app docker image

**Steps to start docker container:**
- Download Docker Desktop (available in Microsoft Store)
- Start Docker Desktop
- Copy .yml file to working directory
- At console in working directory execute "docker compose up -d"
- At Browser type "http://localhost:8501"
- Get your API key at "www.finnhub.io" (takes few seconds)
- Type in API key at front page and push "login"
- Update database at page "Update" (takes one hour)
- Start analyzing stocks
- You can find your saved files under "/workdir/user_data"
